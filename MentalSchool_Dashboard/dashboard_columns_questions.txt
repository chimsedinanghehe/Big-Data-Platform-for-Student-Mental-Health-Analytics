YRBS Mental Health Dashboard - Cot loc, cum chi so, Q va ten cau hoi
Cap nhat theo code hien tai trong app.py va utils/dashboard_core.py

1. Du lieu dang ghep trong dashboard

- Mental School: data/Mental School.csv
  - File co 117 cot raw, trong do co 107 cot Q.
  - Khi preprocess, code strip ten cot va chi giu cac cot co dang q + so: q1 den q107.
  - Sau do them cac cot metadata: Data Source, Population, Study Year, __harmonized_schema.

- HMS:
  - data/HMS_2022-2023_PUBLIC_instchars.csv: header 1620 cot, dashboard doc 103 cot scoped.
  - data/HMS_2023-2024_PUBLIC_instchars.csv: header 1554 cot, dashboard doc 101 cot scoped.
  - data/HMS_2024-2025_PUBLIC_instchars.csv: header 1608 cot, dashboard doc 104 cot scoped.
  - Danh sach scoped trong code gom 118 ten cot. File HMS nam nao co cot nao trong danh sach thi doc cot do.
  - HMS duoc chuan hoa thanh Population = College/University, roi ghep chung voi Mental School.

- Kich thuoc sau khi ghep va preprocess tren du lieu local hien tai:
  - Combined raw: 285,973 dong, 237 cot.
  - Raw analysis sau khi tao Target va loai dong khong co target hop le: 262,854 dong, 120 cot.
  - Cleaned/engineered: 262,854 dong, 155 cot.

2. Cac cot bo loc sidebar dang dung

- Population: tach Middle/High School va College/University.
- Data Source: tach Mental School va tung file/nam HMS.
- q1 - Age.
- q2 - Gender.
- q3 - Grade.
- Target: Lower Risk hoac At Risk.

3. Target mental health risk

- Target = At Risk khi q26 == 1 hoac q84 >= 4.
- Mental School:
  - q26 = Felt sad or hopeless.
  - q84 = Poor mental health days.
- HMS:
  - q26 la proxy tu dep_any/dep_maj, neu thieu thi dung deprawsc >= 10.
  - q84 la proxy severity tu deprawsc va anx_score; lay muc nghiem trong cao nhat giua depression/anxiety.
- Target nay la chi bao nghien cuu mo ta, khong phai chan doan lam sang.

4. Feature model dang dung

Model khong dua toan bo Q vao Random Forest. Feature model hien tai chi gom:

- q1 - Age.
- q2 - Gender.
- q3 - Grade.
- Family Pressure Index.
- Academic Pressure Index.
- Peer & Safety Stress Index.
- Trauma Exposure Index.
- Substance Coping Risk Index.
- Lifestyle Recovery Deficit.

Cac Q mental health truc tiep nhu q26, q27, q28, q29, q30 va q84 khong duoc dung lam input model de tranh leakage. q26 va q84 chi dung de tao Target.

6. HMS pseudo-Q duoc tao de ghep chung voi Mental School

- q1 - Age:
  - HMS source: age.
  - Chi giu age 18 den 80; outlier set missing.
  - Map ve bucket: 18 -> code 7, 19-20 -> code 8, 21-24 -> code 9, 25-34 -> code 10, 35+ -> code 11.

- q2 - Gender:
  - HMS source: sex_birth, gender_female, gender_male, gender_nonbin, gender_queer, gender_trans, gender_transm, gender_transf, gender_selfID, gender_selfid, gender_prefnoresp.

- q3 - Grade:
  - HMS source: yr_sch.
  - Map vao nhom college year: College 1st year, 2nd year, 3rd year, 4th+ year, Graduate/Professional, Other college.

- q26 - Felt sad or hopeless:
  - HMS proxy: dep_any hoac dep_maj; fallback deprawsc >= 10.

- q27 - Considered suicide:
  - HMS source: sui_idea.

- q28 - Made suicide plan:
  - HMS source: sui_plan.

- q29 - Attempted suicide:
  - HMS source: sui_att.

- q84 - Poor mental health days:
  - HMS proxy: deprawsc va anx_score, chuyen ve thang 1 den 5 theo severity.
